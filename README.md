## Easy Installation

1 - Go to your IDE's preferences, select Editor -> Color Scheme, 

2 - Click on the cog icon and select 'Import Scheme', 

3 - Select Toybox Colors.jar from disk

4 - Enjoy a fresh new look for your IDE!
